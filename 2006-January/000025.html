I'd like to put in about twenty votes fo ReStructured Text.

It's simple and flexible and you can generate both HTML & PDF (from LayTex).

I do screenshots using irfanview (paste into it and then ctrl-R to resize), and use the image directive.

The HTML output will be much nicer.

You could even use rest2web to autobuild contents pages and *optionally* a sidebar - and put the adsense stuff into the template for an automated build. HTML is not my strong point though.

I *really* like rest and it's kind of a standard as Python text markups go. What advantage would Asciidoc have over it ?

You would lose your nice index though (until I add indexing to rest2web) - a google search box would sort that (and you get adsense income from search pages too).

If you move to rest then you've got a volunteer to help migrate content. :-)

Having version control on the docs is *good*.
